## 詳細

### 機能要件

### 非機能要件

### ユースケース

主要なユースケースやシナリオを箇条書き、もしくはユースケース図（Marmaid記法等）で記載する。

### 画面イメージ

画面設計やワイヤーフレームがあれば記載する。

### API仕様

APIエンドポイントやパラメータ、レスポンス例などを記載する。

### データ構造
