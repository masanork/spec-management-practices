# サンプル仕様書

---
## タイトル
ユーザー管理API仕様書

## 概要
本ドキュメントは、ユーザー管理機能のAPI仕様について記載する。

## 詳細
### 機能一覧
- ユーザー登録
- ユーザー情報取得
- ユーザー情報更新

### API仕様
| メソッド | パス | 概要 |
| ------- | ---- | ---- |
| POST | /api/users | ユーザー新規登録 |
| GET | /api/users/{id} | ユーザー情報取得 |
| PUT | /api/users/{id} | ユーザー情報更新 |
